Hipervínculos en HTML

Los hipervínculos permiten a los usuarios navegar entre páginas web, documentos, y otros recursos. A continuación, se presentan los diferentes tipos de hipervínculos que se pueden utilizar en HTML:

Hipervínculo a Otra Página Web

El tipo más común de hipervínculo es el que lleva a otra página web. Se utiliza el elemento <a> con el atributo href:

Visitar Google

Hipervínculo a un Correo Electrónico

Se puede crear un enlace que permita enviar un correo electrónico utilizando el prefijo mailto: en el atributo href:

Enviar correo a ejemplo@correo.com

Hipervínculo a un Número de Teléfono

Para enlazar a un número de teléfono, se utiliza el prefijo tel: en el atributo href:

Llamar al +1234567890

Hipervínculo a un Archivo

Los enlaces también se pueden utilizar para descargar archivos. Simplemente se especifica la URL del archivo en el atributo href:

Descargar archivo PDF

Hipervínculo a una Parte Específica de la Misma Página

Es posible crear enlaces que lleven a una parte específica de la misma página utilizando anclas. Se define una ancla con el atributo id y luego se enlaza a ella con el símbolo #:
